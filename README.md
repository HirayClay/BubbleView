# BubbleView
简单的气泡控件，包装内容成气泡形式展现，在通讯应用中比较常见。在某些情况下需要气泡的显示不仅仅是显示在某个控件下方，
有时是需要根据气泡的尖角来控制气泡位置，所以本空间提供了两种显示位置的方式：1）针对控件位置显示气泡  2）针对某点显示气泡
